  ```
  cd /Users/gusiyang/Desktop回车
  python ex1.py
  ```
  或
  ```
  python /Users/gusiyang/Desktop/ex1.py
  ```
+ 这里在运行phthon文件时把文件放到了桌面，这样路径比较简单
+ 如果看到关于ASCII编码的错误，就在python脚本的最上面加入`# -- coding: utf-8 -`
